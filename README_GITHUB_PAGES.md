# GitHub Pages Publishing Notes

This folder is a GitHub Pages-ready static bundle for the VetSOAP transcript findings report.

## Contents

- `index.html`
- `andy_analysis_data.js`
- `andy_analysis_batch2.js` through `andy_analysis_batch18.js`
- `Andy_1700_Transcript_Analysis.md`
- `Point4_Transcript_Evaluation.html`

## Important

- Raw transcript text files are intentionally not included in this public bundle.
- The hosted report keeps the findings, charts, and appendix browser, but transcript links are removed.

## Publish on GitHub Pages

1. Create or choose a GitHub repository.
2. Upload the contents of this folder to the repository root, or to a `docs/` folder.
3. In GitHub:
   - open `Settings`
   - open `Pages`
   - set the source to:
     - `Deploy from a branch`
     - branch: `main`
     - folder: `/ (root)` if files are in the repo root, or `/docs` if they are in `docs/`
4. Save.
5. GitHub will generate a Pages URL.
